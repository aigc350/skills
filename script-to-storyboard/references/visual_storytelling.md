# Visual Storytelling Reference - 视觉叙事参考

## 核心原则

### 1. 每个镜头都有一个目的
- 推动剧情
- 揭示角色
- 建立氛围
- 提供信息

### 2. 视觉叙事 > 对话
- "Show, don't tell" 原则
- 用画面表达情感和关系
- 减少依赖对话

### 3. 节奏控制
- 镜头时长 = 情绪强度
- 紧张时短镜头快速切换
- 沉思时 长镜头停留

---

## 叙事工具

### 视线引导
通过以下方式引导观众视线：
- 焦距
- 运动
- 前景物体
- 眼神方向
- 光线

### 框架表达

| 框架方式 | 情绪效果 |
|----------|----------|
| 开放式框架 | 自由、可能性 |
| 封闭式框架 | 限制、压抑、囚禁 |
| 负空间 | 孤独、缺失、期待 |
| 对称构图 | 平衡、稳定、正式 |
| 三分法 | 自然、舒适 |

### 高度与权力

| 相机位置 | 权力关系 |
|----------|----------|
| 低角度仰拍 | 角色显得强大、有威胁 |
| 高角度俯拍 | 角色显得弱小、无力 |
| 眼平线 | 平等、中性 |

## 常见叙事手法

### 1. 匹配剪辑 (Match Cut)
相同物体/动作在不同场景间切换
**效果**: 连接概念而非时间

### 2. 意象重复 (Motif Recurrence)
关键物体/动作反复出现
**效果**: 强调主题、情感积累

### 3. 冷暖色调对比
暖色调 vs 冷色调
**效果**: 表达情感变化、好坏对比

### 4. 开放式结尾
镜头不完整，切到别处
**效果**: 悬念、紧张感

### 5. 长镜头 (Long Take)
单镜头内完成复杂调度
**效果**: 临场感、紧张、连贯性

## 场景类型叙事

### 对话场景
```
WS 建立关系 → OTS 正反打 → CU 反应 → MCU 对话
```

### 追逐场景
```
ELS 建立 → MS 跟随 → ECU 紧张细节 → 快速剪辑
```

### 情感高潮
```
慢动作 → CU 大特写 → 静止 → 缓慢拉远
```

### 悬疑场景
```
阴影 → 负空间 → 低角度 → 缓慢推近
```

---

## 短剧叙事要点

1. **前三秒法则**: 开头三秒必须抓住观众
2. **每5秒一个高潮点**: 保持节奏紧凑
3. **情绪反转**: 定期改变情绪方向
4. **视觉钩子**: 每15-30秒有新视觉刺激
