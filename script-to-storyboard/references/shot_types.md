# Shot Types Reference - 景别参考

## 景别分类

### 极远景 (Extreme Long Shot - ELS)
**定义**: 也称为建立镜头，展示大环境，人物在画面中很小
**用途**:
- 建立场景位置
- 展示宏大场面
- 强调人的渺小

**示例**: 沙漠、城市天际线、海洋

---

### 远景 (Long Shot - LS)
**定义**: 人的全身，通常保留少量环境
**用途**:
- 展示角色完整姿态
- 包含一定环境信息
- 动作场景常用

---

### 全景 (Wide Shot - WS)
**定义**: 包含角色的完整身型及周围环境
**用途**:
- 建立镜头
- 展示角色与环境关系
- 舞蹈/动作编排

---

### 中远景 (Medium Long Shot - MLS)
**定义**: 膝盖以上
**用途**:
- 两人对话
- 轻度动作场景

---

### 中景 (Medium Shot - MS)
**定义**: 腰部以上
**用途**:
- 标准对话镜头
- 日常场景
- 访谈节目常用

---

### 中近景 (Medium Close-Up - MCU)
**定义**: 胸部以上
**用途**:
- 强调表情
- 对话反应镜头
- 情感场景

---

### 特写 (Close-Up - CU)
**定义**: 脸部特写
**用途**:
- 强调情绪
- 关键反应
- 角色间眼神交流

---

### 大特写 (Extreme Close-Up - ECU)
**定义**: 眼睛、嘴唇、手等细节
**用途**:
- 强调关键细节
- 紧张/悬疑氛围
- 产品/道具展示

---

### 双人镜头 (Two Shot - 2S)
**定义**: 两人同时在画面中
**用途**:
- 对话双方
- 展示关系
- 减少正反打

---

### 过肩镜头 (Over-the-Shoulder - OTS)
**定义**: 越过一人肩膀拍摄另一人
**用途**:
- 标准对话镜头
- 建立空间关系
- 增加深度感

---

## 景别选择指南

| 场景类型 | 常用景别 |
|----------|----------|
| 对话 | MS, MCU, OTS |
| 动作 | LS, WS, ECU |
| 悬疑 | CU, ECU, 负空间 |
| 情感 | CU, ECU |
| 建立 | WS, LS, ELS |
| 追逐 | LS + ECU 交替 |

---

## 景别与情绪

| 景别 | 情绪效果 |
|------|----------|
| ECU | 紧张、亲密、强调 |
| CU | 亲密、参与 |
| MCU | 平衡、交流 |
| MS | 中性、观察 |
| WS | 客观、疏离 |
| LS/ELS | 宏大、孤独、无力 |
