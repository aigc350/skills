# Camera Movement Reference - 运镜参考

## 基本运动类型

### 静止 (Static)
**定义**: 相机固定不动
**用途**:
- 稳定性场景
- 对话场景
- 观察性镜头

**情绪效果**: 客观、冷静、观察

---

### 摇镜 (Pan)
**定义**: 相机绕固定点水平旋转
**用途**:
- 跟随移动物体
- 建立环境
- 展示宽阔空间

**参数**:
- 速度: 慢/中/快
- 方向: 左/右

---

### 倾镜 (Tilt)
**定义**: 相机绕固定点垂直旋转
**用途**:
- 从上到下展示建筑
- 跟随垂直移动
- 强调高度

---

### 推镜 (Dolly In / Push In)
**定义**: 相机向主体移动
**用途**:
- 强调重要性
- 增加紧张感
- 从环境聚焦到细节

**情绪效果**: 强调、接近、紧张、揭露

---

### 拉镜 (Dolly Out / Pull Out)
**定义**: 相机远离主体
**用途**:
- 展示环境
- 结束场景
- 揭示新信息

**情绪效果**: 远离、释放、揭示、孤独

---

### 跟踪 (Track / Dolly)
**定义**: 相机跟随主体移动
**用途**:
- 跟随角色行走
- 追逐场景
- 展示移动中的关系

---

### 升降 (Crane)
**定义**: 相机在垂直方向运动（通常用设备）
**用途**:
- 从地面升起到全景
- 戏剧性建立镜头
- 强调宏大感

---

### 变焦 (Zoom)
**定义**: 改变焦距（不移动相机）
**用途**:
- 快速聚焦
- 模拟推拉效果
- 特效（如眩晕效果）

**注意**: 变焦不等于移动相机，会压缩/扩展背景

---

### 手持 (Handheld)
**定义**: 相机手持拍摄，轻微晃动
**用途**:
- 紧张/追逐场景
- 纪录片风格
- 真实感/临场感

**情绪效果**: 紧张、不稳定、真实、临场

---

## 组合运动

| 运动组合 | 用途 |
|----------|------|
| Dolly + Pan | 弧形运动，跟随角色转弯 |
| Tilt + Dolly | 从脚到头的人像展示 |
| Crane Up + Dolly Out | 升起并拉远，宏大揭示 |

---

## 运动与情绪

| 运动 | 情绪效果 |
|------|----------|
| 慢推 (Slow Push) | 紧张逼近、揭露 |
| 快拉 (Fast Pull) | 震惊、释放 |
| 手持 (Handheld) | 紧张、混乱、真实 |
| 缓慢跟踪 (Slow Track) | 沉思、悬疑 |
| 快速推进 (Fast Dolly) | 攻击性、强调 |
| 荷兰角 (Dutch Angle) | 不稳定、紧张、醉意 |

---

## 运镜规则

### 180度规则
在同一场景中，保持相机在动作轴线同一侧，避免空间混乱。

### 30度规则
角度变化应大于30度，否则剪辑时会有跳跃感。

### 运动连续性
- 运动方向应与剪辑方向一致
- 避免无目的的运动

---

## 短剧常用运镜

| 场景 | 推荐运镜 |
|------|----------|
| 对话 | 轻微手持 + OTS |
| 紧张/悬疑 | 慢推、手持晃动 |
| 动作 | 快速切换、POV |
| 情感高潮 | 慢动作、静止 |
| 转场 | 快拉 + 切入 |
