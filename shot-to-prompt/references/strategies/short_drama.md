# Short Drama Strategy - 短剧爆点策略
# 抖音/快手短剧的内容策略，让内容具有传播性和爆点

## 短剧核心公式

```
短剧爆点 = 冲突 × 节奏 × 情绪 × 反转
```

## 爆点类型

### 1. 打脸爽点 (Face-slapping)
```
公式：弱者在场 → 强者羞辱 → 身份揭晓 → 反转打脸
情绪曲线：压抑 → 爆发 → 爽
Prompt 注入：
- "众人的嘲笑声戛然而止"
- "他的表情从嘲讽变成惊恐"
- "所有人都在等待他的反应"
```

### 2. 身份逆转 (Status Reversal)
```
公式：被看不起 → 真实身份揭晓 → 全场震惊
情绪曲线：轻视 → 揭秘 → 震惊
Prompt 注入：
- "手机屏幕亮起：老陈来电"
- "来人恭敬地称呼：少爷"
- "刚才还嘲讽的眼神变成了恐惧"
```

### 3. 甜宠时刻 (Sweet Moment)
```
公式：暧昧 → 突破 → 撒糖
情绪曲线：升温 → 确定 → 甜
Prompt 注入：
- "四目相对的瞬间"
- "他下意识护住了她"
- "全场安静，只听到心跳声"
```

### 4. 热血逆袭 (Underdog Victory)
```
公式：被压制 → 绝地反击 → 胜利
情绪曲线：绝望 → 爆发 → 畅快
Prompt 注入：
- "他抬起眼，眼神如刀"
- "这一刻，所有屈辱化为力量"
- "王少下意识后退一步"
```

### 5. 悬疑揭秘 (Mystery Reveal)
```
公式：铺垫 → 暗示 → 揭晓
情绪曲线：好奇 → 紧张 → 恍然大悟
Prompt 注入：
- "老管家的称呼暴露了一切"
- "这通电话让所有人变了脸色"
- "她终于明白了一切"
```

## 节奏设计

### AABA 节奏模板
```
A (Establish) - 2秒 - 建立场景和人物
A (Compress) - 2秒 - 压缩冲突
B (Escalate) - 2秒 - 情绪升级
A (Climax) - 2秒 - 爆发/爽点
```

### 镜头组合策略
```
开场 (Hook): 情绪冲击 + 悬念
发展: 递进式压迫
高潮: 打脸/逆转/揭秘
收尾: 余韵 + 留悬念
```

## Prompt 注入关键词

### 情绪爆发关键词
- 瞬间凝固
- 空气突然安静
- 所有人屏住呼吸
- 脸色骤变
- 倒吸一口凉气
- 瞳孔地震

### 装逼打脸关键词
- 刚才还嘲讽的表情
- 笑容僵在脸上
- 不可置信地看着
- 所有人都在等待
- 轮到你了

### 身份揭晓关键词
- 来电显示
- 老陈/老王称呼
- 管家的称呼
- 恭敬的态度
- 少爷/小姐

## 连续性管理

### Shot 连续性检查
1. **表情变化链**: neutral → tension → climax → release
2. **场景连贯**: 同一场景内道具/服装不变
3. **情绪递进**: 每 shot 情绪强度递增

### 跨 Shot 爽点设计
```
Shot 3: 打脸前 - "你说完了？"
Shot 4: 打脸中 - 手机响起
Shot 5: 打脸后 - 全场死寂
```

## 应用规则

1. **识别 narrative_function**
   - `climax` → 注入最强爽点
   - `reveal` → 身份揭晓策略
   - `conflict` → 打脸/对抗策略

2. **emotion 强度映射**
   - `humiliation` → 打脸爽点
   - `shock` → 震惊揭秘
   - `confidence` → 装逼时刻

3. **Shot 顺序优化**
   - 爽点 Shot 前加入压迫 Shot
   - 揭秘 Shot 前加入铺垫 Shot
