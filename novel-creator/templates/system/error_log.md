# 异常日志

记录详细的错误信息。

---

## 错误类型分类

| 类型 | 说明 | 处理方式 |
|------|------|----------|
| CANON_CONFLICT | 违反 canon 设定 | 返回 writer 重写 |
| QUALITY_FAILED | 质量检查多次失败 | 人工介入 |
| CONTEXT_OVERFLOW | 上下文溢出 | 自动裁剪 |
| LOGIC_ERROR | 剧情逻辑错误 | 返回 writer 重写 |
| SYSTEM_ERROR | 系统错误 | 记录并提示 |

---

## 日志格式

```markdown
## 错误记录 #ERR_{ID}

时间：{时间戳}
章节：{章节号}
错误类型：{类型}
错误描述：{详细说明}
上下文：
  - {相关内容}
处理状态：{待处理/已处理/已忽略}
处理方式：{如何解决}
```
