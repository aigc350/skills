# 扩展设定

用于记录运行过程中新增的角色、势力、地点、规则等长期设定。

---

## 使用规则

1. 本文件属于"设定层"，不是 state
2. 所有内容必须追加（不可修改/删除）
3. canon 与 canon_ext 冲突时：
   - 默认以 canon 为准
   - 若存在 CHANGE，则以 CHANGE 为准
4. writer 必须同时参考 canon/* 和 canon_ext.md
