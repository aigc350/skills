# 测试用例 001

玄幻题材长篇小说测试用例，模拟用户 prompt 过程。

---

## 用户 Prompt 模拟

### Prompt 1：创建小说

```
创建一部玄幻小说，名字叫"山海灵印"

简介：边陲少年陆沉身负上古守护者血脉"灵印"，被镇民视为异类，踏上修行之路后结识青玄宗圣女清月、北境蛮族虎莽、南疆蛊师苏婉儿，四人组成团队历经宗门试炼、秘境探险、王朝纷争与种族大战，逐步揭开父母身世真相，最终与企图重定天道的上古魔尊夜穹展开终极对决。

目标200万字，1000章，每章1800-2200字

风格：爽文流，快节奏

主角：陆沉，落霞镇孤儿，上古守护者血脉继承者

核心队友：
- 清月：青玄宗圣女，天才剑修
- 虎莽：北境蛮族少年，部落少主
- 苏婉儿：南疆蛊师，苗疆公主

反派：夜穹，上古魔尊

主线：从被排斥的异类成长为守护天下的英雄
```

---

## 预期 AI 响应

AI 应从上述 prompt 中提取以下信息：

### 基本信息
- 小说名：山海灵印
- 题材：玄幻
- 目标字数：200万字
- 目标章节：1000章
- 章节字数：1800-2200字

### 写作配置
- 节奏类型：快
- 叙事视角：第三人称（默认）
- 写作风格：爽文流

### 角色信息
- 主角：陆沉（落霞镇孤儿，上古守护者血脉继承者）
- 核心配角：清月、虎莽、苏婉儿
- 反派：夜穹（上古魔尊）

### 剧情主线
从被排斥的异类成长为守护天下的英雄

---

## 测试命令序列

### 命令 1：init

**用户输入**：
```
/novel-creator init
```

**AI 询问 & 用户回答模拟**：

```
AI：小说名称是什么？
用户：山海灵印

AI：选择小说题材
用户：玄幻

AI：目标篇幅规模？
用户：长篇（1000章）

AI：选择整体风格基调
用户：爽文流

AI：如何设定主角和主线？
用户：AI推荐

AI：主角名称和身份？
用户：陆沉，落霞镇孤儿，上古守护者血脉继承者

AI：核心队友有哪些？
用户：清月-青玄宗圣女，虎莽-北境蛮族，苏婉儿-南疆蛊师

AI：反派是谁？
用户：夜穹，上古魔尊

AI：主线方向？
用户：从被排斥的异类成长为守护天下的英雄
```

### 命令 2：plan

**用户输入**：
```
/novel-creator plan
```

### 命令 3：run 1

**用户输入**：
```
/novel-creator run 1
```

**验证点**：字数是否在 1800-2200 范围内

### 命令 4：run 3

**用户输入**：
```
/novel-creator run 3
```

**验证点**：连续写作的一致性

### 命令 5：revise

**用户输入**：
```

```

### 命令 6：validate

**用户输入**：
```
/novel-creator validate
```

### 命令 7：export

**用户输入**：
```
/novel-creator export
```
