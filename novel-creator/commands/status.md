# status 命令

显示当前项目状态。

## 命令格式

```
/novel-creator status
```

## 执行步骤

1. 读取 `novel.config.md` 确定当前小说路径
2. 读取关键状态文件：
   - `{当前小说路径}/canon/meta.md` - 基础信息
   - `{当前小说路径}/state/state.md` - 当前状态
   - `{当前小说路径}/state/arcs.md` - 剧情进度
   - `{当前小说路径}/state/chapter_plan.md` - 剩余规划
3. 计算统计数据：
   - 已完成章节数
   - 已写字数
   - 完成进度
4. 格式化输出状态报告

## 输出格式

```
========================================
项目状态: 《小说名》
========================================

【基础信息】
小说名: xxx
题材: xxx
目标章节: xxx 章
目标字数: xxx 字

【进度信息】
已完成章节: xx 章
已写字数: xx 字
完成进度: x.x%

【当前状态】
当前章节: xx
当前地点: xxx
当前时间: xxx
当前冲突: xxx

【系统状态】
运行状态: 正常
剩余规划: x 章
========================================
```

## 错误处理

- 配置文件不存在：提示先使用 init 初始化
- 当前小说目录不存在：提示检查配置
