# list 命令

列出所有小说项目。

## 命令格式

```
/novel-creator list
```

## 执行步骤

1. 读取 `novel.config.md`
2. 解析小说列表
3. 格式化输出

## 输出格式

```
========================================
小说项目列表
========================================
ID: xuanhuan_001    名称: 逆天邪神    题材: 玄幻    状态: 连载中 *
ID: scifi_002       名称: 星际迷航    题材: 科幻    状态: 连载中
ID: urban_003       名称: 都市修仙    题材: 都市    状态: 已完结
========================================
当前活动: 逆天邪神 (xuanhuan_001)
========================================
```

注：`*` 标记当前活动小说

## 错误处理

- 配置文件不存在：提示先使用 init 初始化
