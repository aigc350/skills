# 写作规则

Writer 和 Editor 共用。

---

## 规则级别与扣分

| 标记 | 含义 | Writer | Editor扣分 |
|------|------|--------|------------|
| 🔴 | 强制 | 必须避免 | 发现1处 → 该项0分，建议重写 |
| 🟡 | 警告 | 尽量避免 | 每处扣2分 |
| 🟢 | 提示 | 建议避免 | 每处扣1分 |

---

## 一、词汇类（满分10分）

### 禁止表达 🔴

**Writer**：以下词汇禁止使用

**Editor**：发现任意1处 → 词汇类0分

| 类型 | 禁止词汇 |
|------|----------|
| 过渡词 | 然而、随着时间的推移、不禁、竟然、显而易见 |
| 情态词 | 不得不说、简直、实在是 |
| 眼神描写 | 他的眼中闪过一丝...、她的眼神变得... |
| 心理描写 | 她的心猛地一紧、他感到一阵... |
| 比喻词 | 仿佛...一般、像是...一样 |

---

## 二、结构类（满分10分）

### 排比堆砌 🔴

**定义**：连续3句以上相同句式结构

**判定**：句首词相同 + 语法结构相同

**Writer**：写作时打乱句式，或合并为一句

**Editor**：发现1处 → 结构类0分

```
❌ 他想起了童年。他想起了故乡。他想起了那些逝去的人。
✓ 童年、故乡、那些逝去的人，在他脑海里闪过。
```

### 工整起承转合 🟡

**定义**：章末有总结或升华

**判定**：章末出现抽象概念（成长、意义、命运、人生）

**Writer**：用动作或对话收尾，避免总结

**Editor**：每处扣2分

```
❌ 这一刻，他终于明白了什么是真正的成长。
✓ 他关上门，没再回头。
```

---

## 三、表达类（满分10分）

### 直白象征 🔴

**定义**：直接解释象征物含义

**判定**：象征物 + "就像/仿佛" + 抽象概念

**Writer**：保留象征物本身，删除解释

**Editor**：发现1处 → 表达类0分

```
❌ 茶有苦有甜，回甘很长，就像人生一样。
✓ 茶还是那个味道。淡淡的，回甘很长。
```

### 情绪速成 🟡

**定义**：情绪转变缺乏铺垫

**判定**：情绪词前少于2句具体描写

**Writer**：情绪前增加动作/细节描写

**Editor**：每处扣2分

```
❌ 听完这句话，她感动得热泪盈眶。
✓ 她端着杯子的手顿了一下。低头喝了一口，水已经凉了。
```

### 内心独白泛滥 🟢

**定义**：连续"我"字开头的心理陈述

**判定**：连续3句以上"我/他"+心理动词

**Writer**：打乱句式，或删除"我"字

**Editor**：每处扣1分

```
❌ 我知道他在撒谎。我知道他不想让我知道真相。我不知道该怎么办。
✓ 他在撒谎。真相藏在那些话后面。我能做什么？
```

---

## 四、叙事类（满分10分）

### 全知视角入侵 🟡

**定义**：限知视角中出现上帝视角解释

**判定**：第一人称 + "其实/原来" + 角色不知道的事

**Writer**：删除"其实/原来"，或改为角色可知内容

**Editor**：每处扣2分

```
❌ 其实他不知道，这个决定将改变他的一生。
✓ 他做出了决定。之后的很多年，他都会想起这个下午。
```

### "意识到"泛滥 🟢

**定义**：过度使用认知动词

**判定**：每章超过2次"意识到/发现/明白"

**Writer**：用动作或细节替代认知动词

**Editor**：超标后每处扣1分

```
❌ 他意识到自己一直都在逃避。
✓ 逃避。这个词在他心里放了很多年。
```

## Editor 评分速查表

| 检查项 | 满分 | 扣分规则 |
|--------|------|----------|
| 词汇类 | 10分 | 🔴发现1处=0分 |
| 结构类 | 10分 | 🔴发现1处=0分；🟡每处-2分 |
| 表达类 | 10分 | 🔴发现1处=0分；🟡每处-2分；🟢每处-1分 |
| 叙事类 | 10分 | 🟡每处-2分；🟢每处-1分 |

**通过标准**：每项 ≥6分

---

## Writer 快速检查清单

写作完成后逐项自检：

- [ ] 无禁止词汇
- [ ] 无连续3句相同句式
- [ ] 无直白象征解释
- [ ] 章末无主题升华
- [ ] 情绪转变有铺垫
- [ ] 内心独白不超过段落50%
